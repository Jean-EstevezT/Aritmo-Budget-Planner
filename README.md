# aritmo

A lightweight Electron-based budget planner and dashboard formerly named "Aritmo-Budget-Planner".

Features
- Income and expenses tracking
- Category management
- Budget targets per category
- Charts (Chart.js)

Quick start
1. npm install
2. npm start
